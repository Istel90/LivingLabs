
# 기후변화 리스크 설문 협업 플랫폼

React, TypeScript, Vite로 구성된 독립 웹 프로젝트입니다. 별도 Supabase 계정 없이
브라우저의 localStorage에 설문 데이터와 설정을 저장합니다.

## 개발 환경

- Node.js 20 이상
- npm 10 이상

## 시작하기

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 4174
```

브라우저에서 `http://127.0.0.1:4174`로 접속합니다.

## 데이터 저장 안내

데이터는 접속한 브라우저에 저장됩니다. 동일한 컴퓨터와 브라우저에서는 유지되지만,
여러 컴퓨터 사이에서 데이터가 자동 공유되지는 않습니다.
